# Armor Stand #

## About ##
This mod adds two Armor Stand items. This Armor Stand can get be placed and have armor/tools equipped onto it.

You craft the Armor Stand in the Furniture Bench with 3 Wood Trunks of any type. You can craft the lighter variant with 3 Hay bales in the Furniture Bench.

Simply place the Armor Stand and interact while holding a piece of armor or a tool to equip it onto the Armor Stand. To retrieve just interact with an empty hand.
